PyGObject Samples¶

Gtk3からPythonBindingが GObject Introspectionにて自動生成されるようになった。 Gstreamer-1.0も同様。 importの仕方が

import pygtk
import pygst
pygst.require("0.10")

から

import gi
gi.require_version('Gtk', '3.0')
gi.require_version('Gst', '1.0')
from gi.repository import Gtk, Gst

というように結構変わった。 合わせてAPIの構成もいろいろ変わっているのでサンプル集を作ってみる。

Python3.4(Gentoo Linux)で実験。
